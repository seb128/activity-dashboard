# activity-dashboard

A local command-line tool that aggregates recent activity for one *subject* (yourself or one of your direct reports) across GitHub, Launchpad, Jira, and 1-1 meeting notes, and renders a single static HTML report.

Built for personal week-in-review and 1-1 preparation. Runs entirely on your machine with your credentials — no third-party services, no LLM calls.
